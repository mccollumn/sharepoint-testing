## How to start

In order to start using this project. You will have to do the following steps:

1. Clone the project
2. `npm i` - Install all the dependencies
3. Rename `cypress.sample.json` to `cypress.json`
4. Provide the `username`, `password`, and `appUrl`
5. Run `npm test`
